Introduction

Welcome to Nuxt zodI18n documentation.

This module is inspired by Zod Internationalization library, its goal is to provide everything to translate Zod's error messages.

Features

  • Powered by @nuxtjs/i18n and Zod
  • Provide a Global error map for zod see Zod ERROR_HANDLING
